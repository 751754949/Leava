# leave-miniprogram
基于微信小程序的在线请假系统

### 用户身份

1. 学生：提交请假请求，查看自己的请假详情和请假是否被批准等情况
2. 辅导员：管理学生的请假请求，查看学生请假记录
3. 任课教师：接收经辅导员批准的请假记录



### 技术栈：

前端：WXML+WXSS+JS

后端：Python-Flask

数据库：MySQL
